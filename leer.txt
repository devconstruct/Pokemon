Instrucciones para poder iniciar este proyecto
1.- Para crear el proyecto Nov 2024
npx create-expo-app@latest
Si el proyecto da problemas de permisos usar el comando 
sudo chmod -R 775 /ruta/del/proyecto

2.- Dependencias que se deben de instalar
REACT NAVIGATION Version 7
npm install @react-navigation/native
npx expo install react-native-screens react-native-safe-area-context
npm install @react-navigation/native-stack
npm install @react-navigation/bottom-tabs
npm i react-native-vector-icons@10.0.3
npm i react-native-gesture-handler --> este no es necesario por las versiones actuales que se usan

3.- Intalar las dependencias para formar correctamente el menu
npm install @react-navigation/stack
